# spring-boot-scala

# Spring Boot 热部署
使用 springloaded
``````
<build>
    <plugins>
        <plugin>
            <dependency>
                <groupId>org.springframework</groupId>
                <artifactId>springloaded</artifactId>
                <version>1.2.8.RELEASE</version>
            </dependency>
        </plugin>
    </plugins
</build>
``````
